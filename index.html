Adding a New Lab

  1. Create a new project folder at the root (e.g., ai-proj-5) following the existing structure.
  2. Author the lab using the universal template and include README, code, tests, Docker assets, and catalog-info.yaml.
  3. Update this index.html by duplicating a project card and adjusting the content.
  4. Run pytest <project>/tests to ensure clean test results.
  5. Commit the project folder and index.html, then push to the repository.
